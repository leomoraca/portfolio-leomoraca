Proyectos

Botanicare

App móvil colaborativa para proyectos de reforestación. Permite a usuarios registrar el progreso de sus plantas, recibir recordatorios y comunicarse con administradores. Desarrollada como proyecto académico.

Ficachi y Asociados – Módulo ERP

Durante mis prácticas profesionales, contribuí al desarrollo de un módulo de almacenes en un sistema ERP. Automatización del 80% de procesos, optimización de lógica de negocio con SQL, y rediseño de interfaces con ZK bajo arquitectura MVVM. Trabajo ágil con Git, Asana y SonarQube.

COINFAST

Proyecto freelance que me encargaron en fiverr, el frontend de una página para comprar bitcoins en el que utilizo HTML, CSS, JS. En esta página implemento Apis para hacer la conversión de moneda, animaciones CSS y mobile responsive.
